##栈的应用
- 括号是否匹配
```C++
采用两个计数器，一个统计左括号，一个统计右括号
bool isvalid(const string &s){
    int lefter=0;
    int righter=0;
    for(int i=0;i<s.size();i++){
        if(s[i]=='(')lefter++;
        else if(s[i]==')'){
            if(lefter==0)return false;
            else lefter--;
        }
    }
    return lefter==righter;
}
```
