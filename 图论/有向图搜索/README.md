## 基于DFS的拓扑排序

### DFS的顺序
  
  - 前序，就是在递归之前输出访问节点的信息
  - 后序，就是在递归之后输出访问节点的信息
  - 逆后续，就是后序逆置。

### 拓扑排序

  - 原图dfs得到后续，在逆后续中dfs。
  - 实现请参考“强联通分量分解”。
